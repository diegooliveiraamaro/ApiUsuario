# API RESTful em .NET Core 8.0 com PostgreSQL

## Descrição
Esta aplicação implementa uma API RESTful para gerenciar usuários, com operações CRUD completas, integração com Swagger, e suporte para banco de dados PostgreSQL via Docker.

## Requisitos
- .NET Core 8.0
- Docker e Docker Compose
- Visual Studio 2022
- Cliente PostgreSQL (opcional, para consulta ao banco)
